O Que é o DTI Service?

O DTI Service é um sistema interno de automação, desenvolvido para agilizar e padronizar as operações da equipe de TI. A ferramenta centraliza funcionalidades críticas, desde a configuração de novas máquinas até a execução de scripts de manutenção, reduzindo o tempo gasto em tarefas repetitivas e minimizando a chance de erros manuais.

Funcionalidades Principais

Setup Automatizado

Instala e configura softwares essenciais com um único clique, garantindo a padronização dos ambientes de trabalho.

Otimização de Sistema

Executa rotinas de limpeza e otimização para melhorar a performance e a segurança dos computadores.

Gerenciamento de Impressoras

Mapeia e instala impressoras de rede automaticamente, com base na localização do usuário.

Diagnóstico Rápido

Coleta e exibe informações detalhadas do hardware e software da máquina para um suporte mais ágil.

Consulta ao Active Directory

Verifica os grupos de permissão do usuário e do computador no AD para facilitar a gestão de acessos.

Abertura de Chamados

Automatiza a criação de tickets de suporte, pré-preenchendo informações essenciais para a equipe de TI.

Scripts Personalizados

O DTI Service permite a integração de scripts PowerShell customizados, tornando a ferramenta flexível para qualquer necessidade.

AdicionarScript.ps1 

# Exemplo de Script PowerShell para Limpeza de Arquivos Temporários

# Define o caminho da pasta de arquivos temporários do usuário
$tempPath = "$env:USERPROFILE\AppData\Local\Temp"

# Verifica se o diretório existe
if (Test-Path $tempPath) {
    Write-Host "Limpando arquivos em: $tempPath"
    
    # Remove os itens recursivamente e com força
    Get-ChildItem -Path $tempPath -Recurse -Force | Remove-Item -Recurse -Force -ErrorAction SilentlyContinue
    
    Write-Host "Limpeza de arquivos temporários concluída."
} else {
    Write-Host "Diretório de arquivos temporários não encontrado."
}
